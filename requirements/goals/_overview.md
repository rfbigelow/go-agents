# Goals Overview

## What This Project Aims to Achieve

go-agents is a personal Go library for building LLM-based agents. It exists
to eliminate repeated implementation of common agent patterns across projects,
to codify current best practices for agentic development in Go, and to serve
as a vehicle for learning agent development techniques.

## Core Benefits

- **Reusability** — a shared foundation for multiple agent projects, reducing
  boilerplate (G3.1)
- **Codified best practices** — patterns captured in code rather than informal
  knowledge (G3.2)
- **Accelerated learning** — hands-on exploration of agent development
  patterns in Go (G3.3)

## Scope Boundaries

- **Anthropic only** — no multi-provider abstraction (G6.1)
- **Agent applications only** — not a general-purpose LLM client (G6.2)
