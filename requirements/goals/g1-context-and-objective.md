# G1: Context and Objective

## Organizational Context

This is a personal project for an individual developer who builds agent-based
applications using Go. The domain is agentic AI development — building software
systems where an LLM drives autonomous multi-step workflows.

## Why This Project Is Needed

Each new agent project currently requires rebuilding common infrastructure from
scratch: API client setup, conversation loop management, tool dispatch, and
other recurring patterns. There is no reusable foundation that captures current
best practices for agent development in Go.

This project exists to consolidate those patterns into a single library that
can be reused across multiple agent projects, and to serve as a vehicle for
learning agentic development techniques.

## Overall Objective

Build a reusable Go library that provides lightweight abstractions and a
harness for developing LLM-based agents, embodying current best practices
for agentic development patterns.
