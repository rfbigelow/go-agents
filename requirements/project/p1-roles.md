# P1: Roles and Personnel

## Project Roles

### P1.1: Sole Developer

**Assigned to:** Library developer (G7.1).
**Responsibilities:** All project roles — requirements, design, implementation,
testing, documentation, and release. Also the primary library consumer (G7.2).
**Authority:** All technical and project decisions. No external approval required.

## Communication

This is a single-person project (E4.2). Decisions are documented in commit
messages, PR descriptions, and requirements change requests. An AI assistant
(Claude) participates in requirements elicitation and code development.
